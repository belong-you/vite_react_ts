### 依赖安装 `npm install`

### 运行项目 `npm run dev`

### 项目打包 `npm run build`

## 项目目录：

```

├─ index.html

│  │  ├─ audio
│  │  ├─ common  // 公共组件

│  │  ├─ discuss
```
